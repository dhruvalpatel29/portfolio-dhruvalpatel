## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Basic understanding of HTML/CSS/JavaScript (for customization)

### Installation

1. Clone the repository
```bash
git clone https://github.com/dhruvalpatel29/portfolio.git
```

2. Navigate to the project directory
```bash
cd portfolio
```

3. Open `index.html` in your browser
```bash
# On Windows
start index.html

# On Mac
open index.html

# On Linux
xdg-open index.html
```

Or simply double-click the `index.html` file to open it in your default browser.

## 🎯 Customization

To customize this portfolio for your own use:

1. **Update Personal Information**
   - Edit the content in `index.html`
   - Replace images in the `images/` folder
   - Update social media links

2. **Modify Styling**
   - Edit `css/style.css` for custom styles
   - Adjust colors, fonts, and spacing as needed

3. **Add/Remove Sections**
   - Modify the HTML structure in `index.html`
   - Update navigation links accordingly
